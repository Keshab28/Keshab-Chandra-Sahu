What is CAMS?

The Campus Area Management System (CAMS) is an intelligent real-time monitoring solution designed to track and display the occupancy status of various areas across campus facilities.

How It Works

Empty

No people detected in the area

Open

1-5 people - Area is available

Busy

6-15 people - Area is moderately occupied

Closed

More than 15 people - Area is at capacity

Technology Stack

CAMS is built using cutting-edge technologies:

  • Machine Learning powered people detection
  • Flask backend for robust API services
  • SQLite database for efficient data storage
  • Modern HTML5, CSS3, and JavaScript frontend
